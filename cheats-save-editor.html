Mon Bazou Cheats & Save Editor Guide

Mon Bazou does not have built-in console commands (like press `~`). However, the save files are simple text files. This means you can easily edit them to get Infinite Money, max friendship, or unlock items.

Backup Warning: Always make a copy of your save file before editing! If you make a typo, the game might crash or reset your progress.

Method 1: Manual Save Editing (No Mods)

This is the safest method. You don't need to download anything.

  • Locate your Save File:
    Press Win + R, type %appdata%, and hit Enter. Go back one folder level to AppData, then navigate to:
    AppData\LocalLow\Santa Goat\Mon Bazou\
  • Open the File:
    Find SaveSlot_1.json (or whichever slot you use). Right-click and open with Notepad.
  • Edit "PlayerMoney":
    Press Ctrl + F and search for PlayerMoney. Change the number next to it.
    "PlayerMoney": 999999.0,
  • Edit Friendship:
    Search for Friendship_Leo or other names. Set the value to 100.0 to unlock rewards instantly.
  • Save and Play:
    Save the text file. Launch the game. Enjoy being rich.
